# ANT AI Android App

Slim mobile application layer for ANT AI.

## Purpose

Provides:

- AI control interface
- Agent dashboard
- Project management
- APK build monitoring
- Secure release downloads

## Build Requirements

- Android Gradle project
- Kotlin
- Jetpack Compose
- Secure signing configuration

## Build Flow

```
ANT Core
   |
Android App
   |
GitHub Actions
   |
Signed APK Release
```
