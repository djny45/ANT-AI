# ANT AI APK Downloads

## Production Releases

Production APK releases are published through GitHub Releases after:

- Signed release build succeeds
- Security checks pass
- SHA256 checksum is generated

## Current Status

- Build pipeline: Ready
- Security pipeline: Ready
- Production release pipeline: Ready

## Release Format

```text
ANT AI Android
Version: x.x.x
Security: Verified
SHA256: Provided with release
Download: GitHub Release asset
```

Only signed and verified APK files should be published.
